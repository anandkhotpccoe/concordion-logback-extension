Unique Log File Per Test

If a log file exists in the same location and the specification and has the same base file name, but an extension of ".log" then a link to the log file will be placed in the footer of the specification. This will work regardless of whether the tests are being run in parallel or sequentially.

Look for:

  • Note that the console output is much reduced - this is a result of the logback configuration rather than this extension
  • This extension is particularly useful when running tests in parallel, or have a large number of tests and your log file is unusable
